# 1. What is the difference between null and undefined?

JavaScript এ `undefined` তখন পাওয়া যায় যখন কোনো ভ্যারিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু তাকে এখনো কোনো মান দেওয়া হয়নি। অর্থাৎ ভ্যারিয়েবল তৈরি হয়েছে, কিন্তু তার ভ্যালু সেট করা হয়নি।

`null` হলো ইচ্ছাকৃতভাবে সেট করা একটি খালি মান। যখন আমরা চাই যে কোনো ভ্যারিয়েবল আপাতত ফাঁকা থাকবে, তখন আমরা `null` ব্যবহার করি।

# 2. What is the use of the map() function in JavaScript? How is it different from forEach()?

`map()` একটি অ্যারের প্রতিটি উপাদানের উপর কাজ করে এবং একটি নতুন অ্যারে রিটার্ন করে। এটি মূল অ্যারেকে পরিবর্তন না করে নতুন অ্যারে তৈরি করে।

যদি আমরা কোনো অ্যারের প্রতিটি উপাদান পরিবর্তন করে নতুন ফলাফল পেতে চাই, তাহলে `map()` ব্যবহার করা হয়।

অন্যদিকে `forEach()` শুধু প্রতিটি উপাদানের উপর লুপ চালায়, কিন্তু কোনো নতুন অ্যারে রিটার্ন করে না। এটি সাধারণত তখন ব্যবহার করা হয় যখন শুধু কাজ সম্পন্ন করাই লক্ষ্য, রিটার্ন ভ্যালু দরকার নেই।


# 3. What is the difference between == and ===?

`==` কে বলা হয় loose equality। এটি দুইটি ভ্যালু তুলনা করার সময় প্রয়োজনে টাইপ পরিবর্তন (type conversion) করে নেয়।

`===` কে বলা হয় strict equality। এটি ভ্যালু এবং টাইপ – দুটোই এক কিনা তা যাচাই করে। টাইপ আলাদা হলে সরাসরি false রিটার্ন করে।

# 4. What is the significance of async/await in fetching API data?

`async/await` ব্যবহার করলে asynchronous কোড লেখা সহজ এবং পরিষ্কার হয়। এটি Promise ভিত্তিক কাজকে আরও গুছিয়ে লেখার সুযোগ দেয়।

`async` ফাংশনের ভিতরে `await` ব্যবহার করলে প্রোগ্রাম অপেক্ষা করে যতক্ষণ না API থেকে ডেটা আসে। এতে কোড দেখতে অনেকটা সাধারণ synchronous কোডের মতো লাগে এবং বোঝা সহজ হয়।

এছাড়া `try...catch` ব্যবহার করে সহজে error হ্যান্ডেল করা যায়।


# 5. Explain the concept of Scope in JavaScript (Global, Function, Block).

Scope বলতে বোঝায় একটি ভ্যারিয়েবল কোথা থেকে অ্যাক্সেস করা যাবে।

### Global Scope

ফাংশনের বাইরে ডিক্লেয়ার করা ভ্যারিয়েবল গ্লোবাল স্কোপে থাকে। এগুলো প্রোগ্রামের যেকোনো জায়গা থেকে ব্যবহার করা যায়।

### Function Scope

ফাংশনের ভিতরে ডিক্লেয়ার করা ভ্যারিয়েবল শুধুমাত্র সেই ফাংশনের মধ্যেই ব্যবহার করা যায়। বাইরে থেকে অ্যাক্সেস করা যায় না।

### Block Scope

`let` এবং `const` দিয়ে যদি `{}` ব্লকের ভিতরে কোনো ভ্যারিয়েবল ডিক্লেয়ার করা হয়, তাহলে তা শুধু সেই ব্লকের মধ্যেই সীমাবদ্ধ থাকে। ব্লকের বাইরে গেলে সেটি আর কাজ করে না।
